# Documentation

> 自动生成高质量文档 — API 文档、README、变更日志

## 使用方法

在 OpenCode 对话中输入 `@documentation` 激活此技能。

## 支持的文档类型

- **API 文档** - 根据代码自动生成接口说明、参数、返回值
- **README** - 项目介绍、安装指南、使用示例、贡献指南
- **CHANGELOG** - 根据 git history 生成版本更新日志
- **注释补全** - 为缺少注释的函数/类添加文档字符串
- **架构文档** - 系统设计、组件关系、数据流

## 触发条件

当用户提到以下关键词时自动激活：
- "写文档"、"生成文档"、"写README"
- "更新日志"、"changelog"、"版本说明"
- "加注释"、"补文档"

## 输出要求

- 使用 Markdown 格式
- 包含代码示例
- 结构清晰，层级分明
- 遵循项目已有的文档风格
