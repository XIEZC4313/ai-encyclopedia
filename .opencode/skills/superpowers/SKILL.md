# Superpowers

> 完整的开发方法论 — 头脑风暴 → 规划 → TDD → 执行 → 审核

## 使用方法

在 OpenCode 对话中输入 `@superpowers` 激活此技能。

## 工作流

1. **brainstorming** - 需求分析与方案头脑风暴
2. **writing-plans** - 拆分为 2-5 分钟可执行任务
3. **test-driven-development** - 先写测试，再写实现
4. **subagent-driven-development** - 并行子代理执行
5. **systematic-debugging** - 结构化调试
6. **verification-before-completion** - 验证后提交

## 触发条件

当用户提到以下关键词时自动激活：
- "开发"、"实现"、"构建"、"重构"、"优化"
- "写代码"、"做功能"、"改bug"、"加测试"

## 示例

用户：`帮我开发一个用户登录功能`
→ 自动触发 brainstorming → planning → TDD → build → review 全流程
